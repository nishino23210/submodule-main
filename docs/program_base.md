# JBS会社概要

JBSは、社内外の情報システムを支えることを目的とした、架空の総合IT企業です。このドキュメントでは、会社全体の考え方や組織の見取り図をまとめます。

## 会社の特徴

- 顧客課題を整理して、実務に落とし込むことを重視する
- 開発、運用、サポートをつなげて継続的に改善する
- 技術だけでなく、運用しやすさや説明のしやすさも大事にする

## 主な事業イメージ

1. 業務システムの導入支援
2. インフラと運用の整備
3. 社内向けサービスや改善提案
4. 現場に寄り添った保守・サポート

## 全体の見方

- 会社全体では「何を提供するか」を先に見る
- 部署ごとには「どう動くか」「どこを担当するか」を分けて考える
- 個別案件よりも、共通の進め方をそろえることを意識する
